# 专项-Summary

Summary 题型分为 3 种，一种是让你自己从原文找单词填入，另一种是给定单词让你选，还有一种是 C9 之后出现的，给定问题，然你回答最多 2 个单词。无论哪种，解题方法都是先判定空格处词性，然后再根据定位句句意、结构等判断。
